|       | nll                                | polonius                    |
| '1    | part of code(lines)                | set of loans                |
| '2:'1 | '1 is subset of '2(in line number) | '2 is subset of '1(in loan) |
|       | '1 = {#1,#2}                       | '1={Loan 1,Loan 2}          |
|       | '2 = {#1,#2,#3}                    | '2={Load 2}                 |
|       | (#1 denote line 1)                 |                             |
